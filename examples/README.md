## Adding Your Own Example

If you've used TRUGS Agent in a project, we'd love to include it. Each example should have:

1. `AGENT.md` (or `CLAUDE.md` / `.cursorrules`) — the root TRUG/L instructions, customized for your project
2. `folder.trug.json` — machine-readable project index
3. At least one source file with `<trl>` annotations in comments
4. A `README.md` explaining what the example demonstrates

Submit a PR to the `examples/` directory. See [CONTRIBUTING.md](../CONTRIBUTING.md).
